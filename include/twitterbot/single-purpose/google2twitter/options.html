Twitter API keys

To get the consumer + access tokens, do the following:

  • Go to apps.twitter.com on the account you want to link
  • Create a new app. Fill out the required fields with whatever you want.
  • Change the app permissions (Permissions tab) to read+write.
    • If prompted, add a mobile phone number to the account first. You can remove it again after completing these steps.
  • Go to the 'Keys and Access Tokens' tab and create your access tokens.
  • Copy the consumer + access tokens here.
